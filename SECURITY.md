# Security Policy

This repository powers a public GitHub Pages portfolio website.

## Scope

The site is a static portfolio. It does not use a public login system, backend database, payment flow, or user-submitted form.

## Public Content Rule

All files committed to this repository should be treated as public-facing material. Do not commit:

- passwords, tokens, API keys, or private credentials
- identity documents or government ID numbers
- bank account information
- private home address details
- confidential company documents
- sensitive operational data
- non-public livestock or facility records

## Contact and CV Files

Public contact links and CV files should be reviewed before publishing. CV files should avoid unnecessary sensitive personal data.

## External Scripts

Avoid adding third-party widgets, trackers, analytics, or embedded scripts unless they are necessary and trusted.

## Reporting

For portfolio-related issues, contact the portfolio owner through the public contact details listed on the website.
